agent-tools 使用指南

用于统计 Claude Code 等 AI 编程助手的使用数据，自动采集、定时上报。

安装

管理员会提供安装包下载地址，下载后执行：

npm install -g agent-tools-cli-x.x.x.tgz

安装过程会自动完成：

  • 写入 Claude Code hooks，开始采集使用数据
  • 配置服务器地址

安装完成后验证：

agent-tools --version

正常使用

安装完成后无需任何操作，在 Claude Code 里正常使用即可。数据会在后台自动采集并上报到 Dashboard。

首次安装后，打开 Claude Code 执行一次任何操作（编辑文件、运行命令等），等 1–2 分钟后刷新 Dashboard 即可看到数据。

遇到问题？重新安装

遇到任何问题（无数据上报、命令找不到、hooks 失效等），重新安装即可解决绝大多数问题：

npm install -g agent-tools-cli-x.x.x.tgz

重装会自动刷新 hooks 路径、重新配置 cc-switch 通用配置，无需任何手动操作。

仍然有问题？

到 GitHub Issues 反馈，请附上：

  • agent-tools --version 输出
  • 操作系统版本
  • ~/.agent-tools/data/update-log.json 内容
